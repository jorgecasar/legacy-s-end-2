# Suggested Commands

## Linting and Formatting

- `npm run lint`: Run oxlint for static analysis.
- `npm run lint:fix`: Run oxlint and apply fixes.
- `npm run lint:types`: Run tsc for type checking.
- `npm run format`: Run oxfmt to format the codebase.
- `npm run format:check`: Check if the codebase is properly formatted.

## Testing

- `npm run test`: Run all tests (CI and Integration) in `packages/ai-orchestration`.

## Git / Utilities

- `git status`: Check the current state of the repository.
- `git diff HEAD`: See changes since the last commit.
- `rulesync sync`: Sync Rulesync assets (commands, rules, skills, etc.).
- `serena`: Run Serena for semantic analysis (if available).
