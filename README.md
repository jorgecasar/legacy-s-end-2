# 🎮 Legacy's End

**Legacy's End** is a narrative-driven game built with modern web technologies, following **Clean Architecture** principles and a strong focus on AI-assisted development.

## 🚀 Overview

This project serves as a master replication guide and an implementation of a pedagogical game. It explores the lore of the "Monolith" through the eyes of the hero, Alarion, while demonstrating high-quality software engineering standards.

## 🏗️ Core Architecture

- **Clean Architecture**: Decoupled layers (Domain, Use Cases, Infrastructure, UI).
- **Modern Standards**: ES2022+, TC39 Decorators (`accessor`), Lit Web Components.
- **AI-Powered**: Integrated with `rulesync` and `serena` for intelligent development workflows.

## 🛠️ Getting Started

### Prerequisites

- **Node.js**: v24.0.0 or higher.
- **npm**: v10.0.0 or higher.

### Installation

```bash
# Clone the repository
git clone https://github.com/jorgecasar/legacys-ends.git

# Install dependencies
npm install
```

### Development Commands

```bash
# Lint code
npm run lint

# Format code
npm run format

# Type check
npm run lint:types
```

Refer to [**Quick Start and Setup**](./docs/09-getting-started.md) for more details on the development environment and AI tools.
